Итак...

1) Главное особенность нашего сайта - Think идея, котоую человек пишет на своей стене.
2) Пользователь может написать, лайкнуть или "украсть" ИДЕЮ только если зарегистритован.
		Незарегестрированный пользователь может лишь смотреть определенный контент.
3) ИДЕЯ должна:
	3.1) Быть написана грамотно, без оскорблений и мата (ра нарушение бан на несколько дней)
	3.2) Содержать не меньше 50 слов. 
	3.3) Если ИДЕЯ пренадлежит другому человеку, пользователь обязан указать источник.
4) ИДЕЮ можно шейрить, лайкать или дизлайкать.
5) На главной странице будут изображаться топ "ТИНКЕРЫ" и топ ТИНКИ. 
6) Пользователь может выбрать интересные ему жанры тинков, которые будут изображаться на его странице чаще, чем остальные.
7) Пользователи могут общаться друг с другом только под тинками.
8) ТИНКИ могут быть:
	8.1) Публичными (Доступны всем)
	8.2) Френ-тинки (Только для подписчиков)
	8.3) Для определеенной группы пользователей (Могут выбрать сами)
	8.4) Для определеннобо "населения" (то есть дял тех, кому данный жанр интересен. Скажем ТИНК увидят только те, кому интересна романтика или политика)
	8.5) Прайвит. Для одного конкретного пользователя.
9) ТИНК будет иметь опрееленные жанры.
	Примеры: Политический тинк, романтический, тинк по поводу развлечений (кино,игры,театр), тинк искусств, тинк психологический и тд.
10) ТОП из самых худший ТИНКОВ и ТИНКЕРОВ (то есть пользователи или тинки получившие наибольшее количество тинков)
11) Пользователь может сам выбрать цвет, шрифт и стиль тинка. Так же картинку к тинку
12) Премиум аккаунт. Дает дополнительные стили, наклейки к профилю, диаграмму собственных поклонников и тд.
13) Тинк дает возможность писать рассказы и даже книги (несколько тинков, каждая глава - тинк)
14) Тинкер может иметь подписчиков, друзей.
15) В тинке нет групп, только тинкеры
16) Тинкеры могут обьеденяться и писать тинки вместе
